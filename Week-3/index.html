Other considerations • Lloyd's algorithm is deterministic. Given an initial cluster assignment, it will always return the same final clusters.• k is a hyperparameter and must be chosen appropriately. A hyperparameter is different from a parameter in that it is not "learnt from data" but is chosen before the learning begins. One heuristic to choose a value of k is the elbow method.• Initialization plays a key role in obtaining good clusters. Pathological initialization could lead to very poor clusters. In practice, for a given k, multiple runs of K-means with different initializations are performed. The run which yields the smallest objective function value is chosen. 6.
